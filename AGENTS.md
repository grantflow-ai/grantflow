Features include collaborative document editing with CRDT, intelligent document processing (indexing, crawling, RAG), Firebase authentication, and multi-tenant organization management. Uses GCP infrastructure with Cloud Run, PostgreSQL with pgvector, and implements real-time WebSocket collaboration.

## Microservices Overview

### Backend API (`services/backend`)
- **Framework**: Litestar async with msgspec

- **Database**: PostgreSQL with SQLAlchemy 2.0
- **Pattern**: `/organizations/{id}/projects/{id}/applications/{id}`

1. **Indexer** (`services/indexer`): PDF/DOC/HTML extraction → chunks → embeddings
2. **Crawler** (`services/crawler`): Web scraping with rate limiting
3. **RAG** (`services/rag`): AI content generation with OpenAI/Anthropic/Vertex
4. **Scraper** (`services/scraper`): Grant opportunity discovery from NIH/grants.gov

### Real-time Collaboration
- **CRDT Server** (`crdt/`): Hocuspocus WebSocket server with Y.js
- **Editor** (`editor/`): TipTap collaborative editor package

```
                ↓
         RAG Processing → frontend-notifications
```
